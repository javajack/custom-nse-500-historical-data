What makes it useful

  • Survivorship-bias-free. Every bhavcopy NSE ever published is retained, including days when now-delisted names traded. If a stock was on the tape, it's here.
  • Point-in-time membership. 249 monthly rank snapshots of the top 1,000 symbols by 6-month median turnover. Query members(date, index) and get the universe as of that calendar day.
  • Custom indices, not real Nifty. Define your own from the rank window: nifty_50 = ranks 1–50, midcap_150 = 101–250, my_smallcap_bucket = 351–500. Derived at query time; no rebuild.
  • Two formats per day. Original zip (as-NSE-shipped, cmDDMMMYYYYbhav.csv.zip or BhavCopy_NSE_CM_…) and normalized Parquet with unified columns. Pick the one your pipeline prefers.
  • Git-versioned. Byte-deterministic Parquet, immutable zips. Every update is a clean diff.

Quickstart

Direct download of one day's Parquet:

curl -O https://javajack.github.io/custom-nse-500-historical-data/data/parquet/year=2020/month=03/2020-03-20.parquet

Or via the library locally:

git clone https://github.com/javajack/custom-nse-500-historical-data
cd nse-universe
uv sync
uv run python -c "
from datetime import date
from nse_universe import Universe
u = Universe()
print(u.members(date(2020, 3, 20), 'nifty_50'))
"

Schema

Normalized Parquet columns (same for every year — we reconcile legacy and new NSE formats at ingest):

symbol
NSE ticker, EQ series only (e.g. RELIANCE)
date
trading day (DATE)
open/high/low/close
INR, unadjusted for splits
prev_close
previous close as printed by NSE
volume
shares traded (BIGINT)
turnover
rupee turnover (DOUBLE)
trades
trade count (nullable; absent in pre-mid-2011 NSE CSVs)
year / month
hive partition keys

Caveats

  • Prices in Parquet are unadjusted. Use data/actions/{SYMBOL}.parquet (yfinance splits + dividends; ~66% symbol coverage) to compute adjustment factors, or bring your own adjusted price source.
  • Rank is 6-month median turnover, not free-float market cap. For rank 1–200 the difference is ~1–2 names; at rank 450–500 it can be 5–10% different from the real Nifty 500. Fine for factor studies, not for index replication.
  • Membership rebalances monthly, first trading day. Stable between as_of_dates — intra-month churn invisible by design.
