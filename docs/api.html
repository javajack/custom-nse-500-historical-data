Programmatic access

Every file in this archive is reachable at a stable URL. The same tree is served two ways — via GitHub Pages (this site) and via raw.githubusercontent.com for unmetered scripting use.

URL patterns

Raw NSE zips

https://javajack.github.io/custom-nse-500-historical-data/data/raw/YYYY/MMM/cmDDMMMYYYYbhav.csv.zip                    # legacy (≤ ~2024-07-05)
https://javajack.github.io/custom-nse-500-historical-data/data/raw/YYYY/MMM/BhavCopy_NSE_CM_0_0_0_YYYYMMDD_F_0000.csv.zip  # new (≥ 2024-07-08)

MMM is uppercase three-letter month (JAN .. DEC). Day numbers are zero-padded.

Normalized Parquet

https://javajack.github.io/custom-nse-500-historical-data/data/parquet/year=YYYY/month=MM/YYYY-MM-DD.parquet

Hive-partitioned. Columns unified across both NSE formats — see schema. Zstd-compressed, byte-deterministic.

Corporate actions (yfinance snapshots)

https://javajack.github.io/custom-nse-500-historical-data/data/actions/{SYMBOL}.parquet    # e.g. RELIANCE.parquet

Columns: symbol, event_date, kind ∈ {split, dividend}, ratio, source. ~66% symbol coverage (Yahoo doesn't track every NSE delisting).

Examples

curl — grab one day's Parquet

curl -O https://javajack.github.io/custom-nse-500-historical-data/data/parquet/year=2020/month=03/2020-03-20.parquet

Python (pandas + pyarrow)

import pandas as pd

url = "https://javajack.github.io/custom-nse-500-historical-data/data/parquet/year=2020/month=03/2020-03-20.parquet"
df = pd.read_parquet(url)          # direct HTTP read, no local clone needed
print(df.sort_values("turnover", ascending=False).head())

Python — query a range via DuckDB (in-memory, streams over HTTP)

import duckdb
con = duckdb.connect()
con.execute("INSTALL httpfs; LOAD httpfs;")
df = con.execute("""
    SELECT symbol, AVG(close) AS avg_close, SUM(turnover) AS tot_tn
    FROM   read_parquet('https://javajack.github.io/custom-nse-500-historical-data/data/parquet/year=2020/month=*/*.parquet')
    WHERE  date BETWEEN DATE '2020-03-01' AND DATE '2020-03-31'
    GROUP BY symbol
    ORDER BY tot_tn DESC LIMIT 20
""").df()

Bulk — mirror a whole year with wget

wget -r -np -nH --cut-dirs=2 -R "index.html*" \
     https://javajack.github.io/custom-nse-500-historical-data/data/raw/2020/

Local library + FastAPI

For point-in-time membership queries (custom index → symbols on a given date), clone the repo and run the API locally.

git clone https://github.com/javajack/custom-nse-500-historical-data
cd nse-universe
uv sync
uv run uvicorn nse_universe.api.app:app --port 8765

Endpoints (all return JSON):

MethodPathParametersReturns
GET/health—data coverage + counts
GET/indices—list of custom indices
GET/membersdate, indexsymbols in index on that date
GET/ranksymbol, daterank or null
GET/is_membersymbol, date, indexboolean
GET/universe_atdate, limitfull ranked universe
GET/openapi.json—OpenAPI 3.1 spec for client codegen

Example:

curl "http://127.0.0.1:8765/members?date=2020-03-20&index=nifty_50"
# { "date":"2020-03-20", "index":"nifty_50", "as_of_date":"2020-03-01",
#   "count":50, "symbols":["RELIANCE","HDFCBANK",…] }

Generate a Go or Java client from /openapi.json with openapi-generator.

Stability + versioning

  • URL paths are stable. We never rename directories or files.
  • Historical Parquet is immutable. New days append; existing days don't mutate.
  • If a day's raw zip ever changes upstream (NSE republish), the Parquet re-derives on next ingest. The URL stays the same.
  • Schema additions are backward-compatible (new columns, never removals or renames within a format era).

Rate limits + etiquette

GitHub Pages + raw.githubusercontent.com share a generous public CDN with no per-IP limit under normal use. If you're scraping thousands of files, pace yourself (one request every 100ms is fine) and send a descriptive User-Agent so we can help if something goes wrong.
